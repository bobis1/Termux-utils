# Termux-utils
A group of basic utility scripts intended for use on an android phone with termux
## How to use
The first thing that you need to do to use this collection is you need to is to get termux. You can either get it from the Google Play store or from Fdroid. next you either need to install the git package for termux or you need to copy the repository to the home directory of termux. if you are using git then you can clone this repo to your home repo. The script that will launch the utility menu is Start.sh. To run this you can run bash Start.sh while being in the repo by changing directory into it. from here you can use 1-4 to select what you want to run.
### If you have issues you may not have set up the executable permissions up. try to run chmod +x whateverexecutableisgivingtrouble
